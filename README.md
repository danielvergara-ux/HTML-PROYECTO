# Proyecto "Hola Mundo" — Proyecto básico para enseñar a programar

Este proyecto es intencionalmente muy simple. Está pensado para estudiantes que comienzan con HTML, CSS y JavaScript.

- Objetivo: Mostrar un mensaje "Hola Mundo" centrado en la pantalla.
- Qué aprenderás: estructura HTML mínima, cómo aplicar estilos con CSS y dónde poner JavaScript.

Archivos principales:

- `index.html`: la página principal. Ábrela en el navegador para ver el resultado.
- `styles.css`: estilos que centran el contenido usando Flexbox.
- `app.js`: JavaScript mínimo que escribe en la consola y muestra cómo modificar el DOM.

Primeros pasos (rápido):

1. Abre el archivo [index.html](index.html) en tu navegador (doble clic o arrastrarlo al navegador).
2. Abre las herramientas de desarrollador (F12) y revisa la consola — verás "Hola Mundo desde app.js".
3. Edita el texto del `h1` dentro de `index.html`, guarda y recarga la página para practicar.
4. Prueba a cambiar estilos en `styles.css` (color, tamaño, fuente) y recarga para ver los cambios.

Actividades sugeridas para enseñanza:

- Explica la estructura básica: `<!doctype>`, `<head>`, `<body>`.
- Muestra cómo funcionan las rutas relativas (`styles.css`, `app.js`).
- Haz que los alumnos cambien el texto, el color y el tamaño de la letra.
- Pide que modifiquen `app.js` para cambiar el texto usando `querySelector`.
